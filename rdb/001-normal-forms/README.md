# RDB 正規形チートシート

1NF〜6NFまでの全正規形を、具体例つきでまとめたチートシートです。

## コンテンツ

- [正規形チートシート（HTML）](nf_cheatsheet.html)
- [正規化ウォークスルー（HTML）](nf_walkthrough.html) ── 1つのテーブルを段階的に正規化

## 概要

| 正規形 | ひとこと               | 実務での優先度 |
|--------|------------------------|----------------|
| 1NF    | 値をバラせ             | 必須           |
| 2NF    | 部分依存NG             | 必須           |
| 3NF    | 列依存NG               | 必須           |
| BCNF   | 決定項＝キー           | 推奨           |
| 4NF    | 独立多値NG             | 稀に有効       |
| 5NF    | JOINで壊れるな         | 理論・特殊     |
| 6NF    | 全分解（時間変化対応） | 理論・特殊     |
